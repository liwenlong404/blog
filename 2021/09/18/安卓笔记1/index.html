概述

约束性布局

线性布局

LinearLayout

  • MainActivity已在AndroidManifest中注册，并设为LAUNCHER

一、布局

1、LinearLayout线性布局

  1. wrap_content：内容匹配空间
match_parent：匹配父空间

  2. LinearLayout的属性gravity，决定内部元素的位置

2、RelativeLayout相对布局

二、组件

1、TestView跳转

  1. 声明btn
  2. findViewById从布局文件中找到btn，并赋值
  3. 给声明的btn，设置一个点击监听器，参为匿名内部类new view.OnClickListener
  4. 重写onClick方法，Intnet跳转到另一个Activity
@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.activity_main);
    mBtnTestView = findViewById(R.id.btn_testview);
    mBtnTestView.setOnClickListener(new View.OnClickListener() {
        @Override
        public void onClick(View view) {
            Intent intent = new Intent(MainActivity.this, TestViewActivity.class);
            startActivity(intent);
        }
    });
}
